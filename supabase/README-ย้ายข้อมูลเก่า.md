# วิธีย้ายข้อมูลเก่าจาก Google Sheet เข้าระบบ

มี 2 ทางเลือก เลือกตามจำนวนข้อมูล

---

## ทางที่ 1 — กรอกผ่านเว็บ (เหมาะกับข้อมูลไม่เยอะ / ข้อมูลไม่ครบ)

ใช้หน้า **"เพิ่มข้อมูลสัญญา"** ได้เลย — รองรับ **ใส่วันที่ย้อนหลัง** (ช่อง "วันที่ทำรายการ" เลือกวันที่เก่าได้)
ระบบจะสร้างตารางงวดผ่อนให้อัตโนมัติตามวันที่ย้อนหลังนั้น

ข้อดี: ตรวจสอบทีละเคส แก้ไขทีหลังได้ (คลิกชื่อลูกค้า → แก้ไข)

---

## ทางที่ 2 — นำเข้าเป็นชุดด้วย SQL (เหมาะกับข้อมูลเยอะ)

1. เปิด **SQL Editor** ใน Supabase
2. ก๊อปแม่แบบนี้ แล้วแก้ค่าตามข้อมูลใน Sheet (1 เคส = 1 บล็อก `select`)
   เชื่อมร้านค้าด้วย **รหัสร้าน** ได้เลย ไม่ต้องรู้ id:

```sql
insert into contracts
  (contract_no, inv_no, customer_name, phone, shop_id, model, storage,
   condition, origin, device_price, down_percent, commission_percent, doc_fee,
   finance_amount, monthly_payment, term_months, due_day, transaction_date, status, operator)
select
  'S00016PNQ300',                          -- เลขที่สัญญา
  'INV-xxxxx',                             -- เลข INV
  'น.ส. ตัวอย่าง ลูกค้า',                    -- ชื่อลูกค้า
  '0900000000',                           -- เบอร์โทร
  (select id from shops where code = 'AQ S00016'),  -- รหัสร้าน (ระบบหา id ให้เอง)
  'iPhone 13', '128 GB',                  -- รุ่น / ความจำ
  'new', 'th',                            -- สภาพ(new/used) / แหล่ง(th/inter)
  19900, 30, 12, 100,                     -- ราคาเครื่อง / %ดาวน์ / %คอม / ค่าเอกสาร
  13930, 1100, 12, 15,                    -- ยอดจัดไฟแนนซ์ / ค่าเช่า/เดือน / จำนวนเดือน / ชำระทุกวันที่
  '2026-01-15',                           -- วันที่ทำรายการ (ย้อนหลังได้)
  'active',                               -- สถานะ
  'พนักงาน';                               -- ผู้ดำเนินการ
-- คัดลอกบล็อก insert..select ด้านบนซ้ำ สำหรับเคสถัดไป
```

3. ระบบจะ **สร้างตารางงวดผ่อนให้อัตโนมัติ** (จาก trigger)
4. รัน **`select run_daily_update();`** หนึ่งครั้ง เพื่อให้ระบบคำนวณวันล่าช้า + ค่าปรับ ของเคสย้อนหลังทันที

---

### หมายเหตุ
- สถานะที่ใช้ได้: `active`(ผ่อนปกติ) / `closed`(ปิดสัญญา) / `returned`(คืนเครื่อง) / `returned_closed`(คืนเครื่องปิดสัญญา) / `online`
- รุ่น/ความจำ/อาชีพ ฯลฯ ที่ยังไม่มีในระบบ ให้ไปเพิ่มในหน้า "ตั้งค่า" ก่อน (หรือตาราง `options`)
- ข้อมูลที่ขาด เติมย้อนหลังภายหลังได้เสมอ (หน้าลูกค้าทั้งหมด → คลิกชื่อ → แก้ไข)
